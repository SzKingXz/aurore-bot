# AURORE Bot

Advanced Discord Bot | Modular Architecture | XP System | Moderation

## Features

- 34 Comandos Activos
- Sistema de XP + Leaderboard
- Moderacion Completa (Warn, Kick, Ban, Timeout)
- Autoroles (10 Colores + 12 Regiones)
- Sorteos Automaticos
- Anti-spam Integrado
- Base de Datos Persistente
